Circular convolution clc clear all close all x=[1 2 3 4 5 6] h=[1 2 3] l1=length(x) l2=length(h) N=max(l1,l2) y=cconv(x,h,N) disp("circular convolution using built in method") disp(y) x=[x,zeros(1,N-l1)] h=[h,zeros(1,N-l2)] y=zeros(1,N); for m=0:N-1 for n=0:N-1 z=mod(m-n,N) y(m+1)=y(m+1)+x(n+1)*h(z+1) end end disp("circular convolution manually") disp(y) ***** linear convolution********* clc clear all close all x=[1 2 3 4 5 6] h=[1 2 3] l1=length(x) l2=length(h) L=l1+l2 x=[x,zeros(1,L-l1)] h=[h,zeros(1,L-l2)] X=fft(x) H=fft(h) Y=X.*H disp(Y) Y1=ifft(Y) disp(Y1) y2=cconv(x,h) disp(y2)
